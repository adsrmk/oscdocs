# Documentatie voor ontwikkelaars

Deze sectie is bedoeld voor ontwikkelaars, bureaus en technische beheerders die websites op OS Cloud bouwen en onderhouden. De handleidingen combineren praktische implementatiestappen met uitleg over risico's, controles en terugdraaien van wijzigingen.

## Kies een onderwerp

- **Beveiliging:** beperk het aanvalsoppervlak, configureer beveiligingsheaders en beheer updates en sessies.
- **Prestaties:** verbeter caching, afbeeldingen, fonts en Core Web Vitals.
- **SEO:** beheer indexering, sitemaps en gestructureerde gegevens.
- **Toegankelijkheid:** pas WCAG, semantische HTML en ARIA verantwoord toe.
- **Platform:** configureer PHP, Redis en andere runtimefuncties.

::: warning Test wijzigingen eerst
Maak vóór ingrijpende aanpassingen een herstelbare back-up en test wijzigingen bij voorkeur in een stagingomgeving. Documenteer wat je wijzigt en hoe je dit terugdraait.
:::

Start bijvoorbeeld met [PHP configureren](/developers/php), [HTTP-beveiligingsheaders](/developers/HTTP_security) of [PageSpeed Insights](/developers/psi).
