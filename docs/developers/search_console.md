# Google Search Console

[Google Search Console↗](https://search.google.com/search-console/about) (**GSC**) is een gratis tool van Google waarmee je de aanwezigheid van je site in de Google-zoekresultaten kunt volgen, onderhouden en troubleshooten. In tegenstelling tot `robots.txt` of metatags — die alleen instructies aan Google geven — biedt GSC **feedback**: het laat zien hoe Google jouw pagina's daadwerkelijk ziet en waar het tegen fouten aanloopt.

## Waar dient het voor?

GSC volgt zowel je **zoekprestaties** als de **technische gezondheid** van je site. Hiermee kun je:

- Zien welke zoekopdrachten bezoekers naar je site brengen
- Sitemaps indienen voor snellere crawls
- Mobiele bruikbaarheids- en indexeringsproblemen herkennen die je rankings kunnen schaden
