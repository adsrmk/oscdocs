# ERR_CONNECTION_REFUSED

Deze foutmelding betekent dat de server de verbinding actief weigert — meestal door beveiligingsinstellingen, of omdat de server niet bestaat.

De oorzaak kan ook lokaal liggen: je VPN, proxy, antivirus of firewall blokkeert mogelijk de website of poort.

<br>

## Hoe los je dit op?

**Schakel je VPN, proxy, antivirus of firewall tijdelijk uit** en probeer het opnieuw. Werkt de site daarna wel? Dan blokkeerde een van deze diensten de verbinding — pas de instellingen aan om de website toe te staan.
