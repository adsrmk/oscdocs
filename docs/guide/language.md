# Taal instellen

Het platform ondersteunt meer dan **29 talen**, zodat je je diensten kunt beheren in de taal van jouw voorkeur.

<br>

## Taal wijzigen

1. Klik op je **gebruikersavatar** rechtsboven in het dashboard en kies **Mijn account**.
2. Ga naar het onderdeel **Profiel**.
3. Selecteer je gewenste taal in het dropdown-menu **Taal**.
4. Sla je profiel op — de taal wordt direct toegepast.
