# Betaalmethoden

We ondersteunen **debet-/creditcards**, **SEPA-incasso** en **iDEAL** via [Mollie](https://www.mollie.com/).

Alle betalingen worden veilig verwerkt via Mollie. Afhankelijk van de dienst gaat de betaling automatisch of via een betaallink:

- **Abonnementen** — automatisch geïncasseerd (als SEPA-incasso actief is)
- **Facturen** — betaald via een veilige betaallink (iDEAL, kaart of SEPA)

<div class="info custom-block" style="padding-top: 8px">
Terugkerende betalingen worden geïncasseerd op de <b>eerste dag van elke maand</b>.
</div>

<br>

## Hoe werken betalingen?

Nadat je een bestelling plaatst of een factuur ontvangt:

1. Je ontvangt een betaallink per e-mail.
2. Kies je gewenste betaalmethode (iDEAL, kaart of SEPA-incasso).
3. Rond de betaling veilig af via Mollie.

Voor abonnementen via SEPA-incasso worden toekomstige betalingen automatisch geïncasseerd — je hoeft hier verder niets voor te doen.

<br>

## SEPA-incasso (automatische betalingen)

Als je kiest voor SEPA-incasso:

- Geef je toestemming voor automatische, terugkerende betalingen.
- Worden toekomstige facturen automatisch betaald.
- Hoef je zelf niets meer te doen.
