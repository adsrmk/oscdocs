# WordPress-adminwachtwoord resetten

Gebruik je geen Single Sign-On (SSO) — of ben je je WordPress-adminwachtwoord vergeten? Dan kun je het eenvoudig resetten via het **OS Cloud**-paneel.

<br>

## Wachtwoord resetten

1. Ga naar je applicatie.
2. Klik op **Gebruikers** in het tweede linkermenu.
3. Klik op het **kebab-menu** (drie verticale stippen) naast de gebruiker en kies **Bewerken**.
4. Voer een nieuw wachtwoord in en sla het op.
