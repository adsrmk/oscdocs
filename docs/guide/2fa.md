# Tweestapsverificatie instellen

Tweestapsverificatie (2FA) beschermt je account met een tijdelijke code naast je wachtwoord. Een gelekt of hergebruikt wachtwoord is daardoor niet voldoende om in te loggen. Schakel 2FA in voor iedere gebruiker met toegang tot websites, e-mail, DNS of facturatie.

## Voordat je begint

Installeer een authenticator-app die tijdgebaseerde codes ondersteunt, bijvoorbeeld Google Authenticator, Microsoft Authenticator, 1Password of Authy. Controleer of datum en tijd op je telefoon automatisch worden ingesteld; een verkeerde klok kan codes ongeldig maken.

## 2FA inschakelen

1. Klik rechtsboven op je avatar en open **Mijn account**.
2. Ga naar **Tweestapsverificatie** en kies **Inschakelen**.
3. Open je authenticator-app en voeg een nieuw account toe.
4. Scan de QR-code. Kun je niet scannen, gebruik dan de handmatige sleutel als die wordt aangeboden.
5. Voer de actuele zescijferige code in het OS Cloud-paneel in om de koppeling te bevestigen.
6. Bewaar eventuele herstelcodes direct in een beveiligde wachtwoordmanager. Sla ze niet op hetzelfde apparaat op als je enige authenticator.
7. Log uit en test één keer of je met wachtwoord en 2FA-code opnieuw kunt inloggen.

::: danger
Een QR-code of handmatige 2FA-sleutel is geheim. Deel hiervan geen screenshot met collega's of support.
:::

## Problemen met een code

- Wacht op een nieuwe code en voer die in voordat de timer verloopt.
- Controleer of datum, tijd en tijdzone op je telefoon automatisch worden gesynchroniseerd.
- Controleer of je de code van het juiste OS Cloud-account gebruikt.
- Gebruik een herstelcode als je geen toegang meer hebt tot je authenticator en je die code veilig hebt bewaard.

Kun je niet meer inloggen en heb je geen herstelmogelijkheid, neem dan contact op met support. Om je account te beschermen kan een identiteits- en bevoegdheidscontrole nodig zijn voordat 2FA wordt hersteld.
