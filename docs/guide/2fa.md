# Tweestapsverificatie (2FA)

Tweestapsverificatie (2FA) helpt je account extra goed te beveiligen. Zodra je dit hebt aangezet, heb je voortaan zowel je wachtwoord als een tijdgevoelige code nodig om in te loggen.

<br>


## 2FA inschakelen

Volg deze stappen om 2FA aan te zetten:

1. **Ga naar je instellingen:** Klik op je avatar in de rechterbovenhoek van het dashboard en kies **My account**.
2. **Start de activatie:** Scrol naar het gedeelte **Two-factor authentication** en klik op **Enable two-factor authentication**.
3. **Scan de QR-code:** Open je 2FA-app (zoals Google Authenticator) en kies ervoor om een nieuw account toe te voegen. Scan daarna de QR-code die op je scherm verschijnt.
4. **Bevestig de koppeling:** Voer de verificatiecode uit je app in op het dashboard om de installatie af te ronden.
