# Max. grootte e-mailbijlage

De maximale grootte van een e-mail is **75 MB**, inclusief het bericht en eventuele bijlagen.

E-mail is nooit ontworpen om grote bestanden te versturen. Gebruik voor grotere bestanden een bestandsdeeldienst zoals [WeTransfer↗](https://wetransfer.com/) of clouddiensten als Dropbox of OneDrive.
