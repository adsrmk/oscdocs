# Developers

The **Developers** section is dedicated to technical documentation, tutorials, code snippets, and best practices for modern web development. Since we primarily use **WordPress** as our core CMS, this section focuses on improving site performance, security, and reliability within that ecosystem.

You'll find implementation guides, optimisation techniques, and practical solutions to help you build, maintain, and scale stable, high-performing WordPress websites.

Start with the [developer overview](/en-us/developers/) for complete topic-based navigation. Back up before changing configuration, test risky changes on staging, and always document a rollback.

::: info
Code examples are starting points. Confirm that they suit your WordPress version, web server, plugins, and security policy before using them.
:::
