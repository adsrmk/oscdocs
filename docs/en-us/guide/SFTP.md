# Unable to Access FTP

You can normally go to **Advanced → FTP** to create an account and access your website files via FTP. However, for security reasons, our servers **do not support password-based authentication**. You'll need to create an **SSH key** and upload it in the panel to enable secure access.

<div class="tip custom-block" style="padding-top: 8px">
Even when using SSH keys, you still need to create an <b>FTP account</b> — its username is required for authentication when logging in.
</div>
