# Reset WP Admin Password

If you don't use Single Sign-On (SSO) — or have simply forgotten your WordPress admin password — you can easily reset it from the **OS Cloud** panel.

<br>

## Reset the password

1. Go to your application.
2. Select **Users** in the secondary left menu.
3. Click the **kebab menu** (three vertical dots) next to the user and choose **Edit**.
4. Enter a new password and save.
