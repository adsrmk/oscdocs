# Change the OS Cloud panel language

The OS Cloud panel supports multiple languages. Your language choice is personal and changes only the interface for your user account; it does not change websites, mailboxes or another user's settings.

## Change the language

1. Select your user avatar in the top-right corner and choose **My account**.
2. Open **Profile**.
3. Select your preferred language from the **Language** menu.
4. Save the profile.
5. Refresh the page if every component does not update immediately.

## If the language does not change

- Confirm that the profile was saved successfully.
- Sign out and sign in again.
- Open the panel in a private browser window to rule out stale browser data.
- Be aware that product names and technical terms may intentionally remain untranslated.

You can leave the interface in your preferred language when contacting support. For questions about a button or setting, mention which interface language you use.
