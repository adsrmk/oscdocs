# Manage and Upload Files

After adding a new website, open the **File Manager** to manage all your web files. It lets you upload, edit, compress, download, and unpack files with ease.

<br>

## Folder structure

- **`public_html/`** — The root directory of your website. The `www` entry is a shortcut to this folder, so the two can be used interchangeably. This is also where WordPress installs its core files (`wp-admin`, `wp-content`, `wp-includes`).
- **Subdomains** — Each subdomain has its own separate directory inside the primary domain's files.

## Add files

You can add files in two ways:

- **Drag and drop** them directly into the File Manager, or
- Click the **Upload** button in the top-right corner.

## Manage files

Click the **kebab menu** (three vertical dots) next to a file to access additional actions: **Edit permissions**, **Delete**, **Compress**, **Download**, and more.
