# Error 1015: Rate Limited

Rate limiting is a technique used to control the volume of incoming or outgoing traffic to a network, service, or API within a specific timeframe. It helps prevent abuse, ensures fair usage, and protects systems from being overwhelmed by an excessive number of requests.

This message is typically caused by refreshing the page too quickly or making too many requests within a short period. To prevent abuse, a temporary block has been applied which will be **automatically lifted after 10 seconds.**

<div class="tip custom-block" style="padding-top: 8px">
Our system is designed to allow approximately 5 requests per second per user. This ensures fair usage and protects the system against potential overload.
</div>
