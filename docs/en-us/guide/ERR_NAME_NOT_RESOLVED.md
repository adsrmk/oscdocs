# ERR_NAME_NOT_RESOLVED

This error means your browser couldn't find the IP address for the website's domain name. It's usually caused by a DNS issue, an incorrect URL, or a temporary network problem.

<br>

## How to fix it

1. Double-check the URL for typos.
2. Make sure your **nameservers (NS)** and **DNS records** are pointing to the correct IP address.
3. Try flushing your DNS cache or switching to a different network to rule out a local issue.
