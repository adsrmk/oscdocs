# How to block unwanted emails  <Badge type="warning" text="experimental" />

Blocking emails from unwanted sources helps keep your inbox organised, reduces spam, and protects you against potential security threats. It also ensures you only receive relevant, trusted messages — saving time and avoiding unnecessary distractions.

1. Navigate to the [Emails↗](https://oscloud.nl/emails) section.
2. Select the mailbox you'd like to edit.
3. Open the **Spam Settings**.
4. Scroll down to the **Block/Allow List** section.
5. Enter the email address you want to blacklist under the **Block** section.
6. Save the settings and, where possible, send a controlled test message.

Add a complete email address when you only want to block one sender. Block an entire domain only if you are certain you do not expect legitimate mail from any address on that domain.

## Allow a sender again

Remove the address from the block list or add it explicitly under **Allow**. Use the allow list sparingly: an allowed address can still be compromised or spoofed.

::: info
This feature is experimental. A block only affects future messages and does not remove existing mail. Also check the Spam folder and adjust the general [spam settings](/en-us/guide/spam_settings) if required.
:::

Still receiving messages after saving? Check whether the visible sender matches the actual address in the message details. If you [contact support](/en-us/guide/contact-support), include the full message headers where possible.
