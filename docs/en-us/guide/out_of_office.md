# Out of Office

Use the **Out of Office** section to set up automatic email replies when you're away or unavailable. Senders will receive your reply instantly while you're offline.

<br>

## Set up an auto-response

1. **Set the duration** — Pick a start and end date using the calendar picker.
2. **Or set it indefinitely** — Tick **Send until I turn it off** if you'd rather disable the responder manually later.
3. **Write your message** — Add a clear subject line and a message body that lets contacts know you're away.
4. **Activate it** — Tick the **Enabled** checkbox and click **Add** to put the responder live.
