# Forwarders

Email forwarding automatically redirects incoming emails from one address to another — useful for managing multiple accounts and making sure you don't miss important messages.

<br>

## Set up a forwarder

1. Go to the [Emails↗](https://oscloud.nl/emails) section.
2. Select the mailbox you want to manage.
3. Scroll down to the **Forwarders** section.
4. Enter the email address where incoming messages should be forwarded.
