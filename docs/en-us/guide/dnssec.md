# DNSSEC

**DNSSEC (Domain Name System Security Extensions)** adds a security layer to DNS by digitally signing responses. This protects your domain against attacks like cache poisoning, ensuring visitors are sent to the real server — not a fake one.

<div class="info custom-block" style="padding-top: 8px">
DNSSEC must be supported by your registrar and requires additional configuration on their side before it can be activated.
</div>

## Enable DNSSEC

1. Open your website and click **Domains** in the top navigation.
2. Select **DNSSEC** from the left sidebar.
3. Use the **toggle** to activate DNSSEC.
4. Copy the generated keys and parameters and configure them at your registrar.
